# UpSkillCampus-UCT Projects

This repository contains two data science and machine learning projects completed as part of the University of Cape Town (UCT) internship program.

## Projects Overview

### 1. [Agriculture Crop Production Prediction](./Project4_Ag_Prediction%20of%20Agriculture%20Crop%20Production%20In%20India)
This project focuses on predicting crop production and yield in India using machine-learning models.
- **Key Features:** Interactive Streamlit app (`app.py`) for real-time predictions.
- **Models:** Trained production and yield models (`models/*.pkl`).
- **Data & Visualizations:** Processed data with engineered features and exploratory data analysis (EDA).
- **Getting Started:** Navigate to the project directory, install dependencies (`requirements.txt`), and run the Streamlit app.

[View Project Details](./Project4_Ag_Prediction%20of%20Agriculture%20Crop%20Production%20In%20India/README.md)

### 2. [Smart City Traffic Patterns](./Project9_smart-city-traffic-patterns)
This project analyzes and forecasts smart city traffic patterns using machine learning techniques.
- **Key Features:** Predictive modeling to estimate traffic volume based on temporal features and junction characteristics.
- **Script:** The main analysis and prediction script (`traffic_forecasting.py`) performs feature engineering, model training (RandomForestRegressor), and prediction.
- **Data & Visualizations:** Analyzes traffic data and generates plots (traffic over time, feature importance).
- **Getting Started:** Navigate to the project directory, install dependencies, and run the analysis script.

[View Project Details](./Project9_smart-city-traffic-patterns/README.md)
